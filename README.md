# homebrew-cascade

Homebrew tap for [Cascade](https://github.com/Mearman/cascade) — a cross-platform cloud storage filesystem client built in Rust.

## Install

```bash
brew tap mearman/cascade
brew install cascade
```

## What you get

| Binary | Description |
|--------|-------------|
| `cascade` | Cloud storage filesystem CLI — mount, sync, pin, and manage remote files on-demand |

The formula installs pre-built binaries from [GitHub Releases](https://github.com/Mearman/cascade/releases). Builds are available for:

- macOS (Apple Silicon / Intel)
- Linux (x86_64 / ARM64)

## Updating

```bash
brew upgrade cascade
```

New versions are published automatically when changes land on `main` in the Cascade repo. The formula is regenerated by CI with the correct SHA256 checksums for each platform.

## Uninstall

```bash
brew uninstall cascade
brew untap mearman/cascade
```
